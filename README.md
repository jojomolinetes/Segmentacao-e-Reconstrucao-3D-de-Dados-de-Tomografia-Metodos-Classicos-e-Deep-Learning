# Introdução à Segmentação e Reconstrução 3D de Dados de Tomografia: Abordagens Clássicas e Baseadas em Deep Learning
Neste trabalho, foram exploradas ferramentas computacionais para reconstrução de imagens de Microtomografia Computadorizada, segmentação com abordagens clássicas e baseadas de Deep Learning, e análise de propriedades, aplicando-as a dados de criogéis de nanofibrila de celulose.
